# Statechains

Statechains is an idea originally proposed by [Ruben Somsen](https://twitter.com/SomsenRuben). He does a great job of explaining it in his [bitcoin-dev post](https://lists.linuxfoundation.org/pipermail/bitcoin-dev/2019-June/017005.html) and in his [medium article](https://medium.com/@RubenSomsen/statechains-non-custodial-off-chain-bitcoin-transfer-1ae4845a4a39). This is a library in golang with the helper functions needed to support statechains.
